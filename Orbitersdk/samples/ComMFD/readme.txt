================================================================================
STEP 1    OrbiterSound SDK file copying
================================================================================
Before compile, make sure that OrbiterSound SDK files (header & lib) are copied
into the OrbiterSDK folder:
* copy "OrbiterSoundSDK40.lib" from
  "Sound\OrbiterSound_SDK\VESSELSOUND_SDK\ShuttlePB_project\" to
  "Orbitersdk\lib\"
* copy "MfdSoundSDK40.h" from
  "Sound\OrbiterSound_SDK\MFDSOUND_SDK\MFDSound_Example\" to
  "Orbitersdk\include\"
